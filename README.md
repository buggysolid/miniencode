# miniencode

### What

Encode mkv videos in 1280x720 (720p) x265 using ffmpeg.

### Install

```
chmod +x install.sh
./install.sh
```

### Run

```
./encode.sh my-video-file.mkv 8
```

### Docs

install.sh no-args  
encode.sh file-input number-of-threads
